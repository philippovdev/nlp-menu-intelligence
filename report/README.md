# Report Workspace

This folder is the starting point for the final course report.

## Files

- `main.tex`: report skeleton aligned with the course template
- `lit.bib`: bibliography file to grow as related work is collected

## Suggested Workflow

1. Keep working notes and evidence in `docs/course/`.
2. Move stable text and figures into `main.tex`.
3. Add bibliography entries to `lit.bib` as soon as a source is accepted into
   related work.
4. Generate the final PDF from this folder or move the content into Overleaf at
   the end.
