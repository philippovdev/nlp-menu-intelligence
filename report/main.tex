text or menu documents into a structured menu representation. The project is
framed as a pipeline of NLP tasks: menu item categorization, structured field
extraction, and final menu aggregation. The repository is available at
\url{https://github.com/philippovdev/nlp-menu-intelligence}.

\section{Results}

The first measured run now exists for the current heuristic pipeline on the
in-repo v0 sample item set (10 annotated items): category accuracy is 0.9000,
category macro-F1 is 0.8182, and exact-match price and size accuracy are both
1.0000 on this starter subset.

The first classical classifier baseline is now measured on the fixed
`items.v1.jsonl` split using word-level TF-IDF features and Logistic
Regression. It reaches 0.2778 accuracy and 0.2472 macro-F1 on the validation
split, and 0.3333 accuracy with 0.2790 macro-F1 on the held-out test split.

The second classical classifier baseline uses the same TF-IDF feature setup
with a linear SVM (`LinearSVC`). It reaches 0.2778 accuracy and 0.2556
macro-F1 on the validation split, and improves the held-out test result to
0.3889 accuracy with 0.3433 macro-F1.
